# WIP

本地使用的 chatgpt

## 使用方式

1. 注册一个 openai 账号
2. 从 [openai](https://platform.openai.com/account/api-keys) 获取一个 apikey，填入
3. 填入秘钥开始对话

## 代理问题

域名已经被屏蔽，需要代理；开发环境下如果无法透过代理请求，检查你的代理设置里是不是跳过了 127.0.0.1、localhost。
